# GEMINI.md

## Directory Overview

This directory contains documentation related to the Polish Legal API (ISAP/ELI). The purpose is to provide a comprehensive guide for understanding and interacting with the Polish legal database.

## Key Files

*   **`ISAP-relacje-pełny.md`**: This is a detailed guide to the ISAP/ELI system. It covers:
    *   Metadata structure for legal acts.
    *   Types of relationships between legal acts.
    *   API data structure and endpoints.
    *   Implementation of a knowledge graph for legal data.
    *   Practical examples of API usage.

## Usage

The contents of this directory are intended to be used as a reference for developers and researchers who want to work with the Polish legal database. The `ISAP-relacje-pełny.md` file provides all the necessary information to get started with the ISAP/ELI API.
